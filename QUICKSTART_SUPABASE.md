# Supabase Integration - Quick Start (5 Minutes)

## Step 1: Create .env File

In your project directory, create a file called `.env`:

```bash
SUPABASE_URL=https://your-project-id.supabase.co
SUPABASE_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

Replace with your actual values from Supabase.

## Step 2: Find Your Credentials

1. Open https://supabase.com
2. Go to your project
3. Click **Settings** (left sidebar, bottom)
4. Click **API**
5. Copy:
   - **Project URL** → Paste as `SUPABASE_URL`
   - **anon public** key → Paste as `SUPABASE_KEY`

## Step 3: Create Database Table

1. In Supabase, click **SQL Editor**
2. Click **New Query**
3. Copy and paste this:

```sql
CREATE TABLE IF NOT EXISTS attendance (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  name TEXT NOT NULL,
  roll_no TEXT NOT NULL,
  department TEXT,
  timestamp TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX IF NOT EXISTS idx_attendance_roll_no ON attendance(roll_no);
CREATE INDEX IF NOT EXISTS idx_attendance_timestamp ON attendance(timestamp);
```

4. Click **Run**

## Step 4: Install Supabase Library

```bash
pip install supabase python-dotenv
```

Or upgrade everything:

```bash
pip install -r requirements.txt
```

## Step 5: Run the System

```bash
python3 face_attendance_system.py
```

You should see:

```
[SUPABASE] ✓ Connected to Supabase successfully
[DB] Loaded 2 students with encodings
[SYSTEM] Camera initialized...
```

## Step 6: Check Your Data

1. Open Supabase dashboard
2. Click **Table Editor**
3. Select `attendance` table
4. View all your attendance records! ✓

---

## That's It! 🎉

Your system now logs attendance to:
- **Supabase Cloud** (real-time sync)
- **Local CSV** (automatic backup)

Both happen automatically!

## Troubleshooting

| Issue | Solution |
|-------|----------|
| `.env` file errors | Make sure file is named exactly `.env` (not `.env.txt`) |
| "Could not connect" | Check SUPABASE_URL and SUPABASE_KEY are correct |
| "Table doesn't exist" | Run the SQL schema from Step 3 |
| "Module not found" | Run: `pip install supabase python-dotenv` |

## View Attendance

**In Supabase:**
- Table Editor → attendance → See all records

**In CSV:**
- Open `data/attendance.csv`

**With SQL:**
```sql
SELECT * FROM attendance WHERE DATE(timestamp) = CURRENT_DATE;
```
